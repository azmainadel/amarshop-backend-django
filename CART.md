### Cart Management
- View cart and items
- Add item
- Update item
- Remove item
- Calculate prices

### Save for Later
- Save item
- View saved items
- Move saved items to cart

### Additional Features
- Stock validation
- Price calculations
- Error handlings
- User-mapped carts

## APIs for cart module
- `/cart`
- `/cart/add`
- `/cart/update/{item_id}`
- `/cart/remove/{item_id}`
- `/cart/save-item`
- `/cart/saved-items`
- `/cart/saved-items/{item_id}/move`
